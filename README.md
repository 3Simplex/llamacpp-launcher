# Llamacpp-Launcher Perk Card 🕹️
A ShellCheck-certified AI Command Center for managing local LLM instances.

## Features
- **Nixified V14 Logic**: Automated dependency injection for jq, curl, and bc.
- **Blackwell-Aware**: Integrated with `llamacpp-flake` for GPU offloading.
- **Preset Management**: Remembers context limits and layer counts to avoid OOM.
- **HuggingFace Integration**: Built-in model downloader.
